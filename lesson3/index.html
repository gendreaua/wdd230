Site and Domain Name

The Site Name is: Sunshine and Rainbows

The Domain Name is: snrweather.com

The purpose of this site is to share the current weather patterns and what can be expected for the or following week. This way people can plan their weeks out better, especially any days off they have. It is a way to be better prepared and enjoy the day instead of wondering what the weather would be like.
